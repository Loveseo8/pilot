# Отчет по мини-проекту Habit Pilot

## Цель

Доработать React-проект и показать практическое использование `useEffect`, работы с API, маршрутизации и общего состояния между компонентами.

## Реализованные возможности

1. `useEffect`
   - В `App.jsx` используется для сохранения массива привычек в `localStorage`.
   - В `IdeasPage.jsx` используется для загрузки данных с внешнего API при открытии страницы.

2. Работа с API
   - Страница "Идеи из API" выполняет `fetch` к `https://dummyjson.com/todos?limit=6`.
   - Полученные идеи отображаются в карточках.
   - Если API недоступен, показываются резервные идеи и сообщение об ошибке.

3. Routing
   - Подключен `react-router-dom`.
   - Реализованы две страницы:
     - `/` — список привычек, форма, статистика, поиск и фильтр.
     - `/ideas` — идеи привычек, загруженные с API.
   - Переход между страницами сделан через `NavLink`.

4. Sharing State
   - Общее состояние `habits` хранится в компоненте `App`.
   - Состояние и обработчики передаются дочерним компонентам через props.
   - На странице API можно добавить идею в общий список привычек, после чего она появляется на главной странице.

## Дополнительные улучшения
