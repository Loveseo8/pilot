# Habit Pilot

Небольшое React-приложение для отслеживания привычек.

## Что реализовано

- `useEffect`: сохранение привычек в LocalStorage и загрузка идей с API.
- API: страница "Идеи из API" получает данные с `https://dummyjson.com/todos?limit=6`.
- Routing: две страницы через React Router: "Мои привычки" и "Идеи из API".
- Sharing State: список привычек хранится в `App`, передается в страницы через props, а идеи из API можно добавить в общий список.
- Дополнительно: поиск, фильтрация, аккуратный адаптивный интерфейс.

## Запуск

```bash
npm install
npm run dev
```
